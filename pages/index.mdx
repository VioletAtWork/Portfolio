# Violet Catsburg - Portfolio

Hallo, mijn naam is Violet. Ik ben 25 jaar en heb al mijn hele leven interesse gehad in computers.
Dit is terug te zien in de projecten die ik gedaan heb. Deze zijn hier beneden te zien.

Project 1: Stoplichten
In dit project heb ik geleerd over JavaScript. Dit heb ik gedaan doormiddel van een kruispunt na te bouwen met HTML, CSS en JavaScript. Dit project heb ik gedaan op de Hogeschool Rotterdam.

Project 2: Harry Potter Punten
Ook dit project heb ik gedaan op de Hogeschool Rotterdam. Hierbij moest ik in een klein groepje een app bouwen met PHP en Bootstrap om een schooluitje van de lagere school te voorzien van een app om punten bij te houden.

Project 3: Bioscoop Reserveringssysteem
Nog een project van de Hogeschool Rotterdam. In dit project heb ik in een groep een reserveringssysteem ontwikkeld voor een bioscoop. Dit was om SCRUM te leren en C#.

Project 4: Vlaardingen 1018 Mini-games
Tijdens mijn stage bij Platypus heb ik in Unity mini-games ontwikkeld voor een AR app waarbij gevierd werd dat Vlaardingen 1000 jaar bestaad.

Project 5: Onderzoek AR
In mijn stage bij OmniVR heb ik meer onderzoek gedaan naar AR en de mogelijkheden hiervan binnen een bedrijf. Hierbij heb ik ook kennis gemaakt met het ontwikkelen van VR applicaties.

Project 6: Kapperszaak WiFi
Op de Hogeschool Utrecht Open-ICT heb ik een project gevolgd om een kapperszaak te moderniseren. Hierbij heb ik onderzoek gedaan naar networking en heb een WiFi setup ontworpen.

Project 7: Teddybeer die praat
Ook op de Hogeschool Utrecht heb ik een project gedaan waarbij TI en AI bij kwam kijken. Namelijk om een teddybeer te maken die tegen kinderen kan praten om problemen bij kinderen thuis te kunnen herkennen.
